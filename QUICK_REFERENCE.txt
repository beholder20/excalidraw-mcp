   - Element validation (Zod schemas)
   - SVG rendering orchestration

   - 3 implementations: File (dev), Memory (Vercel), Redis (Upstash)
   - Diagram elements + camera state persistence
   - 18-char truncated UUIDs for checkpoint keys

3. Transport Layer (src/main.ts)
   - HTTP (Streamable) → stateless per-request
   - stdio → Claude Desktop connections
   - Express middleware (CORS + routing)

4. Widget Layer (src/mcp-app.tsx)
   - React component wrapping Excalidraw SVG
   - morphdom diffing for streaming updates
   - Fullscreen mode with safe-area insets
